## 📁 Project Overview

We provide the official implementation of **CANDLE**, a framework for lighting-related image restoration under complex ambient illumination.

CANDLE explores how illumination-invariant semantic priors from self-supervised vision foundation models can improve image restoration under severe color shifts, non-uniform lighting, and material-dependent reflectance.

- **[NTIRE 2026] 🕯️ CANDLE: Illumination-Invariant Semantic Priors for Color Ambient Lighting Normalization**  
  [[arXiv](https://arxiv.org/abs/2604.02785)] [[paper](https://arxiv.org/pdf/2604.02785)] [[code](https://github.com/ron941/CANDLE-NTIRE2026)]

  - Uses DINO semantic priors for robust color ambient lighting normalization.
  - Introduces **DINO Omni-layer Guidance (D.O.G.)** for multi-layer semantic feature injection.
  - Uses color-frequency refinement modules to reduce chromatic collapse and detail contamination.
  - Achieves top rankings in the NTIRE 2026 Ambient Lighting Normalization Challenge.

---

> **Abstract**  
> Color ambient lighting normalization under multi-colored illumination is challenging due to severe chromatic shifts, highlight saturation, and material-dependent reflectance.  
> CANDLE observes that self-supervised DINO features remain highly consistent between colored-light inputs and ambient-lit ground truth, making them effective illumination-robust semantic priors.  
> Based on this observation, CANDLE introduces **DINO Omni-layer Guidance (D.O.G.)** to adaptively inject multi-layer DINO features into successive encoder stages, together with color-frequency refinement modules to suppress chromatic collapse and detail contamination.  
> Experiments on ambient lighting normalization benchmarks show strong restoration performance, and CANDLE achieves top rankings in the NTIRE 2026 Ambient Lighting Normalization Challenge.

## 🧠 Method Overview

CANDLE is designed to restore images degraded by complex colored ambient illumination.

The framework is built upon three key ideas:

1. **Illumination-Invariant Semantic Prior**
   - DINO features are used as robust semantic guidance under colored lighting conditions.
   - These features help preserve object structure and scene geometry during restoration.

2. **DINO Omni-layer Guidance (D.O.G.)**
   - Multi-layer DINO features are injected into the restoration network.
   - The guidance provides both high-level semantic consistency and lower-level structural cues.

3. **Color-Frequency Refinement**
   - Additional refinement modules are used to suppress color contamination.
   - This helps reduce chromatic collapse and improve local detail restoration.

## 🍀 Environment Setup

We recommend using a conda environment.

```bash
conda create -n candle python=3.9 -y
conda activate candle
pip install -r requirements.txt
```

Recommended setup:

```text
OS: Ubuntu 22.04
GPU: NVIDIA RTX 3090 24GB / A100
Python: 3.9
CUDA: 12.1
PyTorch: 2.1+
```
